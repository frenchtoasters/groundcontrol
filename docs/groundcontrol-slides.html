Groundcontrol

OpenCode Plugin for Enhanced Workflow Management

Provider allowlists, session exports, delegation tools, background agents

What It Does

Provider Security

Enforces provider allowlists so only approved providers are used.

Session Export

Auto-exports sessions to Markdown for archiving and review.

Delegation Tools

Delegate work to specialized agents and background tasks.

Code Analysis

AST-grep search and replace for safe large refactors.

Workflow Hooks

Keyword detection, comment checking, and notifications.

Slash Commands

/init-deep and /refactor via the slashcommand tool.

Enhanced Workflow

Delegation and export

Built on the OpenCode plugin system with tool and hook registrations.

{ "plugin": ["@frenchtoasters/groundcontrol@latest"], "allowed-providers": ["amazon-bedrock", "openai"] }

Core Tools

delegate_task - delegate to specialized agents.
background_task - launch async tasks.
background_output - poll background task output.
background_cancel - cancel background tasks.
ast_grep_search - AST-based search.
ast_grep_replace - AST-based replace.

Workflow Hooks

Keyword Detector

Detects workflow keywords and injects guidance.

Comment Checker

Checks edits for comment hints and feedback.

Session Notification

Idle session notifications with optional sound.

Task Resume Info

Suggests resume commands for delegated tasks.

Delegate Retry

Retry guidance on delegation tool errors.

Background Agents

Asynchronous delegation with polling and cancellation.

// Launch background task background_task({ description: "Analyze codebase structure", prompt: "Scan repository for architecture patterns", agent: "explore" }); // Poll for results background_output({ task_id: "bg_..." }); // Cancel if needed background_cancel({ task_id: "bg_..." });

Slash Commands

/init-deep - generate hierarchical AGENTS.md guidance.
/refactor - structured refactor workflow.
slashcommand({ command: "/init-deep" }); slashcommand({ command: "/refactor target --scope=module" });

Configuration

Configured in ~/.config/opencode/groundcontrol.json

{ "sessionLogPath": "~/.opencode/groundcontrol-sessions/", "allowedProviders": ["amazon-bedrock", "openai"], "features": { "backgroundAgents": { "enabled": true, "pollIntervalMs": 2000, "maxPolls": 300 } }, "tools": { "astGrep": { "enabled": true, "maxMatches": 500, "maxOutputBytes": 1048576, "timeoutMs": 300000 }, "delegation": { "enabled": true } }, "hooks": { "keywordDetector": { "enabled": true }, "commentChecker": { "enabled": true, "customPrompt": "" }, "sessionNotification": { "enabled": true, "idleDelayMs": 10000, "sound": false }, "taskResumeInfo": { "enabled": true }, "delegateTaskRetry": { "enabled": true } }, "commands": { "initDeep": { "enabled": true }, "refactor": { "enabled": true } } }
